Employee CRUD Application

A modern Spring Boot CRUD application for managing employee records

Project Overview

This project is a simple Spring Boot application that allows you to perform CRUD operations (Create, Read, Update, Delete) on employee records stored in a PostgreSQL database. The application also includes unit and integration tests, Docker setup, and a continuous integration pipeline via GitHub Actions for deployment to AWS EC2.

Technologies Used

  • Spring Boot - Backend framework
  • PostgreSQL - Database
  • Docker - Containerization
  • GitHub Actions - CI/CD pipeline
  • JUnit - Unit and Integration Testing
  • AWS EC2 - Cloud Deployment

How to Run

To run this project locally, follow these steps:

  1. Clone the repository: git clone https://github.com/ahmed-baz/employee-service.git
  2. Navigate to the project folder: cd employee-service
  3. Set up the PostgreSQL database with Docker (check docker-compose.yml for details)
  4. Build the application: ./mvnw clean install
  5. Run the application: ./mvnw spring-boot:run

CI/CD and Deployment

The project is deployed to AWS EC2 via a CI/CD pipeline configured using GitHub Actions. Every push to the master branch triggers the pipeline, which builds and tests the application, prepare a docker image, push it docker hub and deploys it to an EC2 instance for production use.

Access the Application

Once the application is running, open your browser and go to http://localhost:8080 or access it through the deployed AWS EC2 instance URL.
